# Car WebApp 
A modern car discovery web application built with **HTML, CSS, and JavaScript**.  
Users can explore cars, filter by brand, model, and color, add cars to their wishlist, and view detailed information about each car.  
## **Features**
- **Home Page**
  - Hero section with a “Drive Your Dream” banner
  - Smart search for cars
  - Quick links to Wishlist and Explore
- **Explore Cars Page**
  - Filter cars by brand, model, and color
  - Search bar for quick access
  - Car cards displayed in a neat grid
  - “View Details” button to see individual car info
- **Wishlist Page**
  - Add favorite cars
  - View wishlist cars in a grid
  - Back button for navigation
- **About Page**
  - Displays detailed information about selected cars
  - Price, features, and description
- ## Theme & Design
  - Dark theme with **burgundy accents**
  - Elegant buttons and animations
  - Responsive design for all screens
## Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- GitHub Pages for deployment
